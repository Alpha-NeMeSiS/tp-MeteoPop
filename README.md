# TP – Site vitrine Festival MétéoPop

## Objectif
Créer un site vitrine simple pour un festival (MétéoPop) en **HTML/CSS**.
Le site doit respecter une **DA** (violet/orange), utiliser une **vidéo en hero**, et afficher la **programmation** (artistes + horaires).

---

## Pages à réaliser
- `index.html` : page d’accueil (hero vidéo + boutons + extrait programme + infos pratiques)
- `programme.html` : programmation complète (tous les artistes + horaires par jour)
- `info.html` : informations pratiques
- `billetteries.html` : billetterie (tarifs / pass)
- `contact.html` : page contact
